# GDNet
GDNet: A low-light image enhancement network based on Ghost-Block and unique image decomposition

## Recommended Environment:
 python = 3.7
 
 tensorflow-gpu = 1.9.0
 
 numpy = 1.15.4
 
 scipy = 1.2.0
